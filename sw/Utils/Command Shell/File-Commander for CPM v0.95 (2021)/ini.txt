Таблица патчей для файлов FC.INI и FCxx.COM с версии 0.95i
Адрес Описание текущее значение
0000h-0002h Пожалуйста, не меняйте значение INI
0003h 1 = режим USB PIO, 2 = режим USB NKC 00h (деактивирован)
0004h PIO Адрес данных или порта для NKC 0
0005h Данные PIO B 0
0006h PIO A порт управления 0
0007h Порт управления PIO B 0
0008h Показать рамку, см. Текст 0 (текстовый фрейм)
0009h вертикальная черта 'I'
000Ah ветвь снизу вверх '+'
000Bh ветвь слева направо '+'
000Ch Ветвь сверху вниз '+'
000Dh ветвь справа налево '+'
000Eh нижний левый угол '+'
000Fh верхний левый угол '+'
0010h правый верхний угол '+'
0011h правый нижний угол '+'
0012h горизонтальная линия '-'
0013h 01h = очистить экран при выходе 00h (деактивировано)
0014h 00h - 03h, см. Описание 00h (без обратного)
0015h-0018h Строка символов инверсная (оканчивается на $) 85h, 24h, 0.0
0019h-001Ch Символьная строка, обратная (оканчивающаяся на $) 84h, 24h, 0,0
001Dh Маркировка «Файл отмечен» '!'
001Eh Подтверждение с помощью «J» или «Enter» 00h = только J
001Fh Код ключа для "Eject USB" 00h (деактивирован)
0020h 1 = цветовой выход активен 0 (деактивирован)
0021h цвет символа нормальный 07h = белый
0022h выделен символ, а рамка 03h = желтая
0023h Цветовая маркировка символа 02h = зеленый
0024h фон нормальный 40h = синий
0025h Фоновое меню 00h = черный
0026h Фоновая маркировка 70h = белый
0027h цвет при выходе (удалить только BS) 02h = зеленый / черный
