                              DDraw Options Menu
                          (select from graphics mode)
 drawing:
   5 - toggle draw/erase          0 - toggle browse/draw mode
   R - toggle repeat/manual       W - toggle wrap-around
  ^C - clear screen              ^U - refresh screen
   T - text mode, ESC to exit     B - start/stop box draw
   L - start/stop line draw       C - start/stop circle draw
   S - toggle status line        ^F - fill region
  ^Q - quit DDraw

 subpictures:  to enter or display, follow command by 0, 1, or 2)
   ^D - save subpictures to disk      D - load subpictures from disk
   ^B - start/stop subpicture entry  ^R - display subpicture

 printing:                                      +-----------+
   ^P - print bit image   ^O - print graphics   |7    8    9| (or arrow keys
                                       and text |   \ | /   |     in text mode)
 disk I/O:                                      |4 ---+--- 6|
   ^S - save screen       ^L - load screen      |   / | \   |
   ^D - save subpictures   D - load subpictures |1    2    3|
                                                +-----------+
