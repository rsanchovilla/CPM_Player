In order to explore Musimp do the following:

1) Start Musimp by typing at the CP/M prompt:
MUSIMP ALL

2) Set "PAUSE" for the following demo by typing at the "?" prompt:
PAUSE:100;

3) Execute the demo by typing at the "?" prompt:
   (tyme it in uppercase!)
RDS(DEMO,ALL);

4) Exit Musimp
^C
