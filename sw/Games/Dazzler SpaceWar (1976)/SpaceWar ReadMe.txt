Key Mapping on CPM Player

Joy 1 - controls lower left ship

	Cursor keys
	Joy button 1 - keypad 0 - fire
	Joy button 2 - keypad 1
	Joy button 3 - keypad 2
	Joy button 4 - keypad 3

Joy 2 - controls upper right ship

	Left -  A key
	Right - D key
	Up -    W key - increase thrusters
	Down -  S key - decrease thrusters / hyperspace
	Joy button 1 - G Key - fire
	Joy button 2 - H Key
	Joy button 3 - J Key
	Joy button 4 - K Key
