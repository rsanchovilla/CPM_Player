Key Mapping on CPM Player

Joy 2 - controls cannon


	Left -  A key              move cannon to left
	Right - D key              move cannon to rigth
	Up -    W key 
	Down -  S key 
	Joy button 1 - G Key       fire cannon 
	Joy button 2 - H Key
	Joy button 3 - J Key 
	Joy button 4 - K Key       game start
