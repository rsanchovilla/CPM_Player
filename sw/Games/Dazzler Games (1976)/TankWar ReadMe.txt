Key Mapping on CPM Player

Joy 1 - controls upper left tank

	Cursor keys
	Joy button 1 - keypad 0 - fire
	Joy button 2 - keypad 1
	Joy button 3 - keypad 2 
	Joy button 4 - keypad 3 

Joy 2 - controls lower right tank

	Left -  A key
	Right - D key
	Up -    W key 
	Down -  S key 
	Joy button 1 - G Key ----- fire
	Joy button 2 - H Key 
	Joy button 3 - J Key 
	Joy button 4 - K Key 

To start a game press Button 4 on both Joys at 
same time (Key K and Keypad Key 3)
