Key Mapping on CPM Player

Joy 1 - controls right down corner circle

	Cursor keys
	Joy button 1 - keypad 0 - start game
	Joy button 2 - keypad 1
	Joy button 3 - keypad 2 
	Joy button 4 - keypad 3 - reset game

Joy 2 - controls to left cross

	Left -  A key
	Right - D key
	Up -    W key 
	Down -  S key 
	Joy button 1 - G Key ----- start game 
	Joy button 2 - H Key
	Joy button 3 - J Key 
	Joy button 4 - K Key ----- reset game
