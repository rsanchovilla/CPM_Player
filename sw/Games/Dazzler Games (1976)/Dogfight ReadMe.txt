Key Mapping on CPM Player

Joy 1 - controls left plane

	Cursor keys
	Joy button 1 - keypad 0 - fire
	Joy button 2 - keypad 1
	Joy button 3 - keypad 2 --+-- press both for max thurst
	Joy button 4 - keypad 3 --'   press one for medium thrust

Joy 2 - controls right plane

	Left -  A key
	Right - D key
	Up -    W key ------------ push yoke to dive
	Down -  S key ------------ pull yoke to climb
	Joy button 1 - G Key 
	Joy button 2 - H Key
	Joy button 3 - J Key --+-- press both for max thurst
	Joy button 4 - K Key --'   press one for medium thrust
