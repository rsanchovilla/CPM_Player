Cursor movement (mapped to PC keyboard cursor keys on  cpm_player.ini)

	^H	Left
	^B	Right
	^N	Up
	^O	Down

Other keys

	^A	Insert 		Deposit cell at cursor
	^B	Del		Erase cell at cursor

	^F  	Run		Start evolution:

When evolution running

	F 	to freeze
	G 	to resume
	S 	to stop
